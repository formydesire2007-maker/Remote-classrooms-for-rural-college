 # Remote Classrooms for Rural College

## 📖 Overview

Remote Classrooms for Rural College is an online learning platform designed to provide quality education to students in rural areas. The system enables teachers and students to connect virtually through live classes, recorded lectures, study materials, assignments, and communication tools.

The goal of this project is to bridge the educational gap between urban and rural colleges by making learning accessible from anywhere with an internet connection.

---

## 🚀 Features

- 👨‍🏫 Teacher Login & Dashboard
- 👩‍🎓 Student Login & Dashboard
- 📹 Live Online Classes
- 🎥 Recorded Video Lectures
- 📚 Course Materials Upload & Download
- 📝 Online Assignments
- 📊 Attendance Tracking
- 📢 Announcements & Notifications
- 💬 Student-Teacher Communication
- 🔒 Secure Authentication

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Remote-Classrooms-for-Rural-College.git
```

2. Navigate to the project directory

```bash
cd Remote-Classrooms-for-Rural-College
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open your browser

```
http://localhost:5000
```
